# DesignHub 키워드 생성 참고자료

이 문서는 로컬 리서치 보고서 `miricanvas_element_keyword_report (1).md`를 DesignHub 메타데이터용 재사용 규칙으로 정리한 것이다. 공식 랭킹 공식이 아니라 실무 우선순위 프레임으로 사용한다. MiriCanvas 내부 검색 랭킹 가중치는 공개되어 있지 않다.

## 핵심 원칙

키워드는 제작 과정이 아니라 구매자가 검색할 내용을 설명해야 한다.

좋은 키워드 세트는 다음 축을 조합한다.

- 대상 또는 사물
- 요소 형식
- 스타일
- 용도
- 시즌 또는 이벤트
- 대상 사용자 또는 산업
- 색상 또는 무드

가장 효율적인 공식은 다음과 같다.

```text
대상/사물 + 요소 형식 + 스타일 + 용도 + 시즌/이벤트 + 대상/산업 + 색상/무드
```

## 개수와 언어

- 쉼표로 구분한 고유 키워드 20~25개를 사용한다.
- 가장 구체적이고 공식 주제에 가깝고 시각적으로 정확한 단어를 앞쪽에 둔다.
- 사용자가 명시적으로 요청하지 않으면 한 키워드 목록 안에서 한국어와 영어를 섞지 않는다.
- 현재 실무 기준은 25개까지로 보되, 정책이 바뀐 듯하면 실제 DesignHub UI를 다시 확인한다.

## 우선순위 축

느슨한 동의어를 추가하기 전에 아래 축을 먼저 채운다.

| 축 | 질문 | 예시 |
| --- | --- | --- |
| 대상/사물 | 무엇인가? | 화살표, 꽃, 병원, 커피, 강아지, 책, 쿠폰 |
| 요소 형식 | 어떤 asset 형식인가? | 아이콘, 일러스트, 프레임, 스티커, 패턴, 차트 |
| 스타일 | 어떻게 보이는가? | 손그림, 3D, 픽셀, 수채화, 미니멀, 빈티지 |
| 용도 | 어디에 쓰이는가? | 발표, 카드뉴스, 포스터, 썸네일, 메뉴판 |
| 시즌/이벤트 | 언제 유용한가? | 새해, 봄, 여름, 추석, 크리스마스 |
| 대상/산업 | 누가 쓰는가? | 학교, 병원, 카페, 교회, 쇼핑몰 |
| 색상/무드 | 어떤 색감이나 느낌인가? | 파랑, 노랑, 파스텔, 귀여운, 깔끔, 고급 |

## 강한 수요 클러스터

asset과 맞는 경우 아래 클러스터를 우선 고려한다.

- 문서/PPT: 발표, 보고서, 제안서, 차트, 그래프, 표
- SNS 콘텐츠: 카드뉴스, 인스타, 썸네일, 유튜브, 프로필
- 홍보/상업: 이벤트, 세일, 할인, 쿠폰, 배너, 포스터, 상세페이지
- 교육: 학교, 수업, 시간표, 학원, 선생님, 학생
- 시즌: 새해, 봄, 여름, 추석, 수능, 크리스마스, 연말
- 기본 장식: 화살표, 말풍선, 프레임, 아이콘, 배경, 꽃, 하트, 리본, 별, 체크
- 의료/건강: 병원, 의료, 건강, 간호사, 마스크, 약, 상담
- 카페/음식: 카페, 커피, 빵, 디저트, 메뉴, 음료, 레시피

## 피할 것

`keywords`와 `elementName`에서 제작, 파일, 관리, 플랫폼 용어를 제거한다.

```text
Photopea, API, 프롬프트, imagegen, 배경제거, PNG, JPG, SVG, GIF, MP4,
2D, 350DPI, 투명배경, run ID, 날짜, DesignHub, MiriCanvas,
CSV, Premium, 클립아트, 디자인소스, 배경소스, 꾸밈요소
```

## 품질 확인

메타데이터를 채택하기 전에 확인한다.

- 키워드가 20~25개다.
- 한 행 안에서 키워드 중복이 없다.
- 앞쪽 키워드가 제작 과정이 아니라 보이는 asset을 설명한다.
- 대상/사물, 형식, 스타일, 용도, 수요 신호가 포함되어 있다.
- 시즌 키워드가 asset과 제출 타이밍에 맞다.
- 공식 랭킹 요소로 확인되지 않은 내용을 공식 가중치처럼 단정하지 않는다.
